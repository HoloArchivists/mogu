Mogu XML Namespace

You found this page because our RSS feeds contain an URL that links here. Said URL is not an actual page but rather a unique identifier used to prevent name collisions with other XML namespaces.

The namespace contains the following additional, informational tags:

  • <mogu:seeders> holds the current amount of seeders on the respective torrent.

  • <mogu:leechers> holds the current amount of leechers on the respective torrent.

  • <mogu:downloads> counts the downloads the torrent got up to the point the feed was refreshed.

  • <mogu:infoHash> is the torrent's V2 infohash, a unique identifier, in hexadecimal.

  • <mogu:infoHashV1> is the torrent's V1 infohash, a unique identifier, in hexadecimal. This field may be absent.

  • <mogu:categoryId> contains the ID of the category containing the upload in the form category_subcategory.

  • <mogu:category> contains the written name of the torrent's category in the form Category - Subcategory.

  • <mogu:size> indicates the torrent's download size to one decimal place, using a magnitude prefix according to ISO/IEC 80000-13.

  • <mogu:trusted> indicates whether the torrent came from a trusted uploader (YES or NO).

  • <mogu:remake> indicates whether the torrent was a remake (YES or NO).

  • <mogu:comments> holds the current amount of comments made on the respective torrent.
